# Technology Stack: Protik

## Core Framework & Language
*   **Language:** TypeScript - For type safety and better developer experience.
*   **Framework:** React 19 - Leveraging the latest React features for building the UI.
*   **Build Tool:** Vite - For fast development starts and optimized production builds.

## Styling & Layout
*   **Utility CSS:** Tailwind CSS - For rapid UI development with a utility-first approach.
*   **Design System:** zinc-based palette (Zinc 100/950) for a clean, high-contrast industrial look.

## Animation & Interaction
*   **Animation Engine:** Framer Motion - Essential for handling complex scroll physics, parallax effects, and smooth component transitions.
*   **Icons:** Lucide React - A comprehensive set of clean, consistent icons.

## Media & Assets
*   **Video Hosting/Playback:** Mux Player (@mux/mux-player-react) - Used for high-performance video previews and case study content.
*   **Typography:** Google Fonts
    *   `Space Grotesk` (Headings)
    *   `Inter` (Body)
    *   `JetBrains Mono` (Technical details/Monospace)
